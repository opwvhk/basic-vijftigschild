# Basic Computerspellen
Dit zjn de Basic-programmas uit "Basic Computerspellen", geschreven door M.Th.A.M. Vijftigschild,
en uitgegeven in 1982 door Kluwer Software-reeks.

Het boek is nauwelijk meer te vinden, maar de programmacode heb ik bewaard uit sentimentele
overwegingen. Er zijn legio modernere bronnen om mee te leren programmeren.
